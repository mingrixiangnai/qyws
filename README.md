# AstrBot 随机播放文件夹里千原万神小丛雨的语音插件

## 简介

在群里输入【千原万神】即可发送文件夹里的随机语音

## 配置

本插件无需任何配置。

## 注意事项

*   本插件需要 AstrBot 机器人环境。
*   请确保 AstrBot 机器人已正确安装并配置。
*   只测试了QQ。
*   需要把“回复引用”消息关掉。
